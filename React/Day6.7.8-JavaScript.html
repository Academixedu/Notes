console.log(x); // Output: 10 } console.log(x); // Output: 10 (Accessible outside the block because it's function-scoped) } varExample(); ``` #### `let`: Block-scoped, hoisted but not initialized, cannot be re-declared within the same scope - **Scope:** `let` is only accessible within the block it is declared in. - **Hoisting:** `let` is hoisted but not initialized, leading to a `ReferenceError` if accessed before declaration. - **Re-declaration:** `let` cannot be re-declared within the same scope. **Example:** ```javascript function letExample() { if (true) { let x = 10; console.log(x); // Output: 10 (Accessible within the block) } // console.log(x); // Error: x is not defined (Not accessible outside the block) } letExample(); ``` #### `const`: Block-scoped, hoisted but not initialized, cannot be re-declared, must be initialized at declaration, and cannot be reassigned - **Scope:** `const` is only accessible within the block it is declared in. - **Hoisting:** `const` is hoisted but not initialized, leading to a `ReferenceError` if accessed before declaration. - **Re-declaration:** `const` cannot be re-declared within the same scope. - **Initialization:** `const` must be initialized at the time of declaration and cannot be reassigned. **Example:** ```javascript function constExample() { if (true) { const x = 10; console.log(x); // Output: 10 (Accessible within the block) } // console.log(x); // Error: x is not defined (Not accessible outside the block) // const x = 20; // Error: Identifier 'x' has already been declared // x = 20; // Error: Assignment to constant variable } constExample(); ``` ### Summary - **`var`:** Function-scoped, hoisted, can be re-declared. - **`let`:** Block-scoped, hoisted but not initialized, cannot be re-declared within the same scope. - **`const`:** Block-scoped, hoisted but not initialized, cannot be re-declared, must be initialized at declaration, and cannot be reassigned.
