1. **What is CSS and what is its purpose?** - **Answer:** CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It controls the layout, colors, fonts, and overall visual appearance of web pages. 2. **How do you link a CSS file to an HTML document?** - **Answer:** You link a CSS file to an HTML document using the `` element within the `` section of the HTML file. - **Example:** ```html ``` 3. **What are the different ways to apply CSS to a webpage?** - **Answer:** CSS can be applied in three ways: - **Inline CSS:** Using the `style` attribute within HTML elements. - **Internal CSS:** Using a ` ``` 4. **Explain the box model in CSS.** - **Answer:** The CSS box model describes the rectangular boxes generated for elements in the document tree and consists of: `content`, `padding`, `border`, and `margin`. - **Example:** ```css div { width: 100px; height: 100px; padding: 10px; border: 5px solid black; margin: 20px; } ``` 5. **What is the difference between `margin` and `padding`?** - **Answer:** `Margin` is the space outside the border of an element, while `padding` is the space inside the border, between the border and the content. 6. **What are CSS selectors? Can you name a few types?** - **Answer:** CSS selectors are used to select and style elements in HTML. Types of selectors include: - **Element Selector:** `div { color: blue; }` - **Class Selector:** `.className { color: blue; }` - **ID Selector:** `#idName { color: blue; }` - **Attribute Selector:** `[type="text"] { color: blue; }` 7. **How can you include comments in CSS?** - **Answer:** Comments in CSS are included using `/* comment */`. - **Example:** ```css /* This is a comment */ p { color: blue; /* Another comment */ } ``` 8. **Explain the difference between `class` and `id` selectors.** - **Answer:** `class` selectors are used to style multiple elements and are defined with a period (`.`), while `id` selectors are unique to a single element and are defined with a hash (`#`). 9. **What is the difference between `inline`, `inline-block`, and `block` elements?** - **Answer:** - **Inline:** Elements do not start on a new line and only take up as much width as necessary. - **Block:** Elements start on a new line and take up the full width available. - **Inline-block:** Elements are like inline elements, but they can have width and height set. - **Example:** ```html Inline

It can be used to set the space between lines of text. - **Example:** ```css p { line-height: 1.5; } ``` 15. **How do you make text bold, italic, or underlined using CSS?** - **Answer:** - Bold: `font-weight: bold;` - Italic: `font-style: italic;` - Underlined: `text-decoration: underline;` - **Example:** ```css .bold { font-weight: bold; } .italic { font-style: italic; } .underlined { text-decoration: underline; } ``` 16. **Explain the difference between `em` and `rem` units.** - **Answer:** `em` units are relative to the font size of the element's parent, while `rem` units are relative to the font size of the root element (`html`). 17. **What is the difference between `relative`, `absolute`, `fixed`, and `sticky` positioning?** - **Answer:** - **Relative:** Positioned relative to its normal position. - **Absolute:** Positioned relative to its nearest positioned ancestor. - **Fixed:** Positioned relative to the viewport, stays in the same place even when scrolling. - **Sticky:** Switches between relative and fixed depending on the scroll position. - **Example:** ```css .relative { position: relative; top: 10px; } .absolute { position: absolute; top: 10px; left: 10px; } .fixed { position: fixed; top: 10px; } .sticky { position: sticky; top: 0; } ``` 18. **How do you create a two-column layout using CSS?** - **Answer:** Use Flexbox or CSS Grid to create a two-column layout. - **Example with Flexbox:** ```css .container { display: flex; } .column { flex: 1; } ``` - **Example with CSS Grid:** ```css .container { display: grid; grid-template-columns: 1fr 1fr; } .column { padding: 10px; } ``` 19. **What is Flexbox and how do you use it?** - **Answer:** Flexbox is a layout model that allows you to create complex layouts with ease. Use the `display: flex;` property to create a flex container. - **Example:** ```css .container { display: flex; } .item { flex: 1; } ``` 20. **How does CSS Grid differ from Flexbox?** - **Answer:** Flexbox is one-dimensional (either a row or a column), while CSS Grid is two-dimensional (both rows and columns). CSS Grid is more powerful for creating complex layouts. 21. **Explain how the `float` property works.** - **Answer:** The `float` property allows elements to float to the left or right, allowing text and inline elements to wrap around them. - **Example:** ```css .left { float: left; } .right { float: right; } ``` 22. **How do you clear floated elements?** - **Answer:** Use the `clear` property to clear floated elements or use a clearfix hack. - **Example:** ```css .clearfix::after { content: ""; clear: both; display: table; } ``` 23. **How do you create a responsive design using CSS?** - **Answer:** Use media queries to apply different styles based on screen size and other characteristics. - **Example:** ```css @media (max-width: 600px) { body { background-color: lightblue; } } ``` 24. **What are media queries and how are they used?** - **Answer:** Media queries are used to apply different styles for different devices or screen sizes. - **Example:** ```css @media (max-width: 768px) { .container { flex-direction: column; } } ``` 25. **How do you use CSS to hide an element?** - **Answer:** Use the `display: none;` or `visibility: hidden;` properties. - **Example:** ```css .hidden { display: none; } .invisible { visibility: hidden; } ``` 26. **How do you style a link to be different when it is hovered over?** - **Answer:** Use the `:hover` pseudo-class. - **Example:** ```css a:hover { color: red; } ``` 27. **What is a pseudo-class and can you name a few examples?** - **Answer:** A pseudo-class is used to define a special state of an element. Examples include `:hover`, `:focus`, `:active`, `:visited`, and `:first-child`. 28. **What is a pseudo-element and how is it different from a pseudo-class?** - **Answer:** A pseudo-element is used to style specified parts of an element. It is different from a pseudo-class which styles the whole element. Examples include `::before`, `::after`, and `::first-letter`. - **Example:** ```css p::first-letter { font-size: 2em; } ``` 29. **How do you use the `z-index` property in CSS?** - **Answer:** The `z-index` property specifies the stack order of an element. Higher values are placed in front of lower values. - **Example:** ```css .box1 { position: absolute; z-index: 1; } .box2 { position: absolute; z-index: 2; } ``` 30. **What is the difference between `visibility: hidden;` and `display: none;`?** - **Answer:** `visibility: hidden;` hides the element but still takes up space, while `display: none;` removes the element from the document flow entirely. 31. **How do you create a CSS transition?** - **Answer:** Use the `transition` property to specify the duration, timing function, and the property to be animated. - **Example:** ```css .box { width: 100px; height: 100px; background-color: blue; transition: background-color 0.5s ease; } .box:hover { background-color: red; } ``` 32. **What is the difference between CSS animations and CSS transitions?** - **Answer:** CSS transitions are used for simple animations between two states, while CSS animations are more complex and can include multiple keyframes. 33. **How do you create a CSS animation?** - **Answer:** Use the `@keyframes` rule to define the animation and the `animation` property to apply it. - **Example:** ```css @keyframes myAnimation { from { opacity: 0; } to { opacity: 1; } } .box { animation: myAnimation 2s infinite; } ``` 34. **What are `@media` rules in CSS?** - **Answer:** `@media` rules are used to apply styles based on certain conditions, such as screen size or device type. - **Example:** ```css @media (max-width: 600px) { body { background-color: lightblue; } } ``` 35. **What is the `@import` rule in CSS?** - **Answer:** The `@import` rule is used to import external CSS files into a stylesheet. - **Example:** ```css @import url('styles.css'); ``` 36. **How do you set the background image of an element?** - **Answer:** Use the `background-image` property. - **Example:** ```css body { background-image: url('background.jpg'); } ``` 37. **How do you create a gradient background in CSS?** - **Answer:** Use the `background: linear-gradient` or `background: radial-gradient` properties. - **Example:** ```css .gradient { background: linear-gradient(to right, red, yellow); } ``` 38. **What is the difference between `nth-child` and `nth-of-type` selectors?** - **Answer:** `nth-child` selects the nth child of a parent regardless of type, while `nth-of-type` selects the nth child of a specific type. - **Example:** ```css p:nth-child(2) { color: red; } p:nth-of-type(2) { color: blue; } ``` 39. **How do you change the cursor appearance when hovering over an element?** - **Answer:** Use the `cursor` property. - **Example:** ```css a { cursor: pointer; } ``` 40. **What is a CSS preprocessor and can you name a few?** - **Answer:** A CSS preprocessor is a scripting language that extends CSS and compiles it into regular CSS. Examples include SASS, LESS, and Stylus. 41. **How do you write a nested selector in SASS?** - **Answer:** Nested selectors in SASS are written by placing the child selectors inside the parent selectors. - **Example:** ```scss .parent { .child { color: red; } } ``` 42. **What is the difference between `inline` and `block` elements?** - **Answer:** `Inline` elements do not start on a new line and only take up as much width as necessary, while `block` elements start on a new line and take up the full width available. 43. **How do you make a list horizontal?** - **Answer:** Use the `display: inline;` or `display: inline-block;` properties on the `li` elements. - **Example:** ```css ul { list-style-type: none; } li { display: inline; } ``` 44. **How do you style a specific list item using CSS?** - **Answer:** Use the `:nth-child` pseudo-class. - **Example:** ```css li:nth-child(2) { color: red; } ``` 45. **How do you create a fixed header using CSS?** - **Answer:** Use the `position: fixed;` property on the header element. - **Example:** ```css header { position: fixed; top: 0; width: 100%; background-color: lightblue; } ``` 46. **How do you use the `calc()` function in CSS?** - **Answer:** The `calc()` function is used to perform calculations for property values. - **Example:** ```css .box { width: calc(100% - 50px); } ``` 47. **How do you create a drop-down menu using CSS?** - **Answer:** Use the `:hover` pseudo-class to show the drop-down menu when hovering over the parent element. - **Example:** ```css .dropdown { position: relative; display: inline-block; } .dropdown-content { display: none; position: absolute; background-color: white; box-shadow: 0px 8px 16px 0px rgba(0,0,0,0.2); } .dropdown:hover .dropdown-content { display: block; } ``` 48. **What is a media feature in CSS media queries?** - **Answer:** Media features describe the characteristics of the user's device, such as width, height, orientation, and resolution. - **Example:** ```css @media (min-width: 600px) { body { background-color: lightblue; } } ``` 49. **How do you use CSS to make an image responsive?** - **Answer:** Use the `max-width: 100%;` property to make an image responsive. - **Example:** ```css img { max-width: 100%; height: auto; } ``` 50. **How do you apply styles to an element based on its attributes?** - **Answer:** Use attribute selectors. - **Example:** ```css input[type="text"] { border: 1px solid black; } ``` These questions and answers cover a wide range of CSS topics and provide practical examples to illustrate the concepts.
