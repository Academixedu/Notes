### Detailed CSS Tutorial for Beginners This tutorial will guide you through various CSS concepts and how to apply them to an HTML file. You will see the changes in real-time using Chrome's Developer Tools. #### HTML Structure Let's start with a simple HTML file. ```html CSS Tutorial

Welcome to My CSS Tutorial

This is a simple tutorial to learn CSS step-by-step.

Subheading 1

Some content goes here.

Subheading 2

More content goes here.

``` #### CSS Styles Create a CSS file (`styles.css`) and add the following styles step-by-step. Each step will include instructions on what changes to expect and how to verify them using Chrome. ### Step 1: Basic Styles **CSS:** ```css body { font-family: Arial, sans-serif; background-color: #f0f0f0; } ``` **Changes to Expect:** - The font for the entire page will change to Arial or a similar sans-serif font. - The background color of the page will change to a light gray (`#f0f0f0`). **How to Verify:** - Open your HTML file in Chrome. - Right-click on the page and select `Inspect`. - In the `Elements` tab, you should see the `body` element with the applied styles. ### Step 2: Header Styling **CSS:** ```css header { background-color: #4CAF50; color: white; text-align: center; padding: 20px 0; } ``` **Changes to Expect:** - The header background will change to a green color (`#4CAF50`). - The text color will change to white. - The text will be centered, and padding will be added to the top and bottom. **How to Verify:** - Check the `header` element in the `Elements` tab to see the applied styles. - You should see the header with the new background color, text color, and alignment. ### Step 3: Paragraph Styling **CSS:** ```css #intro { color: #333; font-size: 18px; margin: 20px 0; } ``` **Changes to Expect:** - The paragraph with the id `intro` will have a dark gray color (`#333`). - The font size will increase to 18px. - There will be a 20px margin above and below the paragraph. **How to Verify:** - Inspect the `p` element with the id `intro` to see the applied styles. - You should see the changes in color, font size, and margin. ### Step 4: Content Box Styling **CSS:** ```css .content { background-color: #ffffff; border: 1px solid #ddd; padding: 15px; margin: 10px 0; border-radius: 5px; } ``` **Changes to Expect:** - Each `div` with the class `content` will have a white background. - A light gray border will be added around the `div`. - Padding will be added inside the `div`. - There will be a 10px margin above and below each `div`. - The corners of the `div` will be slightly rounded. **How to Verify:** - Inspect one of the `div` elements with the class `content` to see the applied styles. - You should see the changes in background color, border, padding, margin, and border-radius. ### Step 5: Text Alignment and Font Styling **CSS:** ```css h1, h2 { font-family: 'Georgia', serif; text-align: left; } h1 { color: #4CAF50; font-size: 36px; } h2 { color: #555; font-size: 24px; } ``` **Changes to Expect:** - Both `h1` and `h2` elements will have a serif font (Georgia). - The `h1` element will be colored green (`#4CAF50`) and have a font size of 36px. - The `h2` element will be colored dark gray (`#555`) and have a font size of 24px. - Both headings will be left-aligned. **How to Verify:** - Inspect the `h1` and `h2` elements to see the applied styles. - You should see the changes in font family, color, font size, and text alignment. ### Step 6: Additional Styling **CSS:** ```css body { line-height: 1.6; } section { margin: 20px; } ``` **Changes to Expect:** - The line height of the text in the body will increase, making it more readable. - There will be a 20px margin around the `section` element. **How to Verify:** - Inspect the `body` and `section` elements to see the applied styles. - You should see the changes in line height and margin. ### Full CSS File Here is the complete CSS file after all the changes: ```css body { font-family: Arial, sans-serif; background-color: #f0f0f0; line-height: 1.6; } header { background-color: #4CAF50; color: white; text-align: center; padding: 20px 0; } #intro { color: #333; font-size: 18px; margin: 20px 0; } .content { background-color: #ffffff; border: 1px solid #ddd; padding: 15px; margin: 10px 0; border-radius: 5px; } h1, h2 { font-family: 'Georgia', serif; text-align: left; } h1 { color: #4CAF50; font-size: 36px; } h2 { color: #555; font-size: 24px; } section { margin: 20px; } ``` ### Final Verification - Open your HTML file in Chrome. - Use the `Inspect` tool to see how each style is applied to different elements. - Experiment with modifying CSS properties in the Developer Tools to see changes in real-time. This detailed CSS tutorial covers basic styles, font settings, colors, alignment, and additional styling to enhance the appearance of a simple HTML page. Use Chrome's Developer Tools to inspect and verify the applied styles.
